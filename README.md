# RBX Driver — Kernel-mode injector driver for Roblox

## 📋 Overview
This is a Windows kernel-mode driver designed to facilitate memory operations in the RobloxPlayerBeta.exe process. It provides low-level memory management capabilities including allocation, read/write, protection changes, and VAD (Virtual Address Descriptor) manipulation.
